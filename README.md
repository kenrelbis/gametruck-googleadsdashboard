# GameTruck PPC Dashboard

Interactive account performance dashboard for tracking Google Ads performance across GameTruck locations.

## Features

- Quick Paste from Google Ads (auto-parses Cost, Clicks, CPC, Conversions, Cost/Conv)
- Auto health scoring based on Cost/Conv target (≤$20 On Target, $20–30 Attention, >$30 Critical)
- Sortable leaderboard with search & filter
- Visual charts (spend vs conversions, health distribution)
- Per-account printable/PDF reports to share with owners
- Date scope views: Daily, Weekly, MTD, Monthly, Custom
- CSV export for archiving each reporting period
- Data persists in your browser

## Usage

Open `index.html` in any modern browser, or visit the hosted version.
